# Monorepo Startkit

## Setup development

- pnpm add -g pnpm
- pnpm install

## Commands

- `clean` - Removes build folders from all packages
- `lint` - Runs [eslint](https://eslint.org/) checks
- `prettier` - Runs [prettier](https://prettier.io/) checks
- `build` - Builds all packages
- `test` - Tests all packages
